# Futureinterns3
# 📊 CRM Dashboard UI

A modern web-based CRM dashboard designed for small agencies to manage clients, tasks, and daily workflows in one place.

## 🎯 Objective
To simplify client management, task tracking, and follow-ups through a clean and intuitive dashboard.

## 💡 Features
- Dashboard overview (tasks, events, pending work)
- Schedule & activity tracking
- Task and reminder management
- Client/workflow visibility
- Analytics overview
